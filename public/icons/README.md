# Placeholder PWA icons

These icon files are simple SVG placeholders for the Virtual Jap Mala PWA setup. Replace them with high-quality PNG or SVG artwork for your brand.

- `icon-192.svg` — small browser icon
- `icon-512.svg` — large browser icon
- `maskable-icon-512.svg` — maskable install icon
- `apple-touch-icon.svg` — iOS home screen icon placeholder

Recommended replacements:
- use square PNG files for Android/Chrome
- use PNG or SVG with transparent background for iOS
- keep a consistent saffron and dark theme for app icons
